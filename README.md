# 📚 Personal Book Archive & Tracker

An intelligent backend system designed to manage personal libraries, track reading progress, and archive literary insights.

## 🛠️ Tech Stack
- **Language:** Python 3.12
- **Framework:** FastAPI
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy / SQLModel

## 🚀 Key Features
- **Library Management:** Add, edit, and categorize books.
- **Reading Progress:** Real-time tracking of "To Read," "Reading," and "Completed" statuses.
- **Insight Logs:** A dedicated space for reviews and favorite quotes.
- **External Integration:** (Planned) Fetching book metadata via Open Library API.

## 🛠️ Setup Instructions
1. Clone the repo
2. Create virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt`
